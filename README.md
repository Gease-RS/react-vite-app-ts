## Instalação

1. React com Typescript

```bash
yarn create vite <nome_do_projeto> --template <tipo_aplicacao>
yarn create vite react-vite-app-ts --template react-ts
```

1.1 React sem typescript

```bash
yarn create vite react-vite-app-ts --template react
```

2. Instale o projeto

```bash
yarn create vite react-vite-app-ts --template react
```

3. Rodando o projeto

```bash
yarn dev
```
